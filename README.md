## Simple java RMI program for calculating student report.

### How to run the code
1. Open terminal in the root folder of your program.
2. Compile the java code using " javac *.java "
3. Create RMI Stub using " rmic stubFileName "
4. Connect mysql-jar-connector file.
5. Start Server
6. Open another terminal and start Client
7. Enjoy

NB: Execution steps may vary based on the operation system.
